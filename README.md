# WB Reviews

WordPress-плагин для автоматического отображения отзывов с Wildberries на страницах товаров.

## Что делает

- Берёт nmId товара из кастомного поля записи/страницы
- Запрашивает отзывы через [Feedbacks API Wildberries](https://feedbacks-api.wildberries.ru)
- Выводит карусель с отзывами перед блоком `.related-items.can-like`
- Кэширует ответы через WP Transients

## Установка

1. Загрузи папку `wb-reviews` в `/wp-content/plugins/`
2. Активируй плагин в **WP Admin → Плагины**
3. Перейди в **Настройки → WB Reviews** и укажи API-ключ

## Настройка API-ключа

### Вариант 1 — через админку

Настройки → WB Reviews → поле «API-ключ продавца WB».

### Вариант 2 — через wp-config.php (рекомендуется)

```php
define('WB_API_KEY', 'твой_токен');
```

Константа имеет приоритет над значением из админки.

Ключ берётся в личном кабинете WB → **Настройки → Доступ к API**.

## Использование

На нужной записи или странице заполни поле **«🛍 Wildberries nmId»** в правой панели редактора.

nmId — это число из URL товара на Wildberries:

```
https://www.wildberries.ru/catalog/229857564/detail.aspx
                                 ↑
                              nmId
```

## Настройки плагина

| Параметр | По умолчанию | Описание |
|---|---|---|
| API-ключ | — | Токен продавца WB |
| Название custom field | `wb_nm_id` | Поле с nmId в записи |
| Количество отзывов | `20` | Максимум 200 |
| Время кэша | `3600` | В секундах (1 час) |

## Отладка

Ошибки пишутся в `wp-content/debug.log`. Для включения логов добавь в `wp-config.php`:

```php
define('WP_DEBUG', true);
define('WP_DEBUG_LOG', true);
define('WP_DEBUG_DISPLAY', false);
```

Также ошибки видны в консоли браузера (F12) — например, неверный API-ключ выдаст:

```
[WB Reviews] API вернул код 401 для nmId 229857564. Ответ: {"title":"Unauthorized"...}
```

Для просмотра полного ответа API открой вкладку **Network** (F12) и найди запрос к `admin-ajax.php?action=wb_get_reviews`.

## Сброс кэша

- **Автоматически** — при сохранении записи с новым nmId
- **Вручную** — кнопка «Сбросить кэш» в метабоксе редактора
